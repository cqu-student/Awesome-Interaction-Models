## Text Interaction (文本交互)

### 📄 文章与论文

- [Attention Is All You Need](https://arxiv.org/abs/1706.03762) - Transformer 架构的奠基论文，引领了 NLP 领域的革新。
- [BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding](https://arxiv.org/abs/1810.04805) - 双向预训练模型在 NLP 中的应用。
- [Language Models are Unsupervised Multitask Learners](https://d4mucfpksywv.cloudfront.net/better-language-models/language-models.pdf) - GPT-2 论文。

### 💻 开源代码

- [Hugging Face Transformers](https://github.com/huggingface/transformers) - 最受欢迎的 NLP 库，提供了预训练模型和工具。
- [FastText](https://github.com/facebookresearch/fastText) - Facebook 开发的快速文本分类和词向量学习库。
- [spaCy](https://github.com/explosion/spaCy) - 工业级自然语言处理库。
- [OpenAI GPT Repository](https://github.com/openai/gpt-2) - GPT-2 官方实现。

## Image Interaction (图像交互)

### 📄 文章与论文

- [An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale](https://arxiv.org/abs/2010.11929) - Vision Transformer (ViT) 论文。
- [CLIP: Learning Transferable Models for Multimodal Learning](https://arxiv.org/abs/2103.14030) - 开放式视觉语言模型的突破。
- [Diffusion Models Beat GANs on Image Synthesis](https://arxiv.org/abs/2105.05233) - 扩散模型在图像生成中的优势。

### 💻 开源代码

- [PyTorch Vision](https://github.com/pytorch/vision) - PyTorch 官方计算机视觉库。
- [OpenAI CLIP](https://github.com/openai/CLIP) - 多模态学习模型的官方实现。
- [Stability AI Diffusers](https://github.com/huggingface/diffusers) - 扩散模型库，支持文本生成图像等任务。
- [YOLO](https://github.com/ultralytics/yolov5) - 目标检测的实时算法实现。

## Video Interaction (视频交互)

### 📄 文章与论文

- [Towards Good Practices for Very Deep Video Understanding](https://arxiv.org/abs/2102.06183) - 深度视频理解的最佳实践。
- [ViViT: A Video Vision Transformer](https://arxiv.org/abs/2103.15808) - 视频的 Vision Transformer 扩展。
- [Temporal Segment Networks: Towards Good Practices for Deep Action Recognition](https://arxiv.org/abs/1608.00859) - 视频行为识别的经典方法。

### 💻 开源代码

- [PyTorchVideo](https://github.com/facebookresearch/pytorchvideo) - Meta 开发的视频理解库。
- [MMAction2](https://github.com/open-mmlab/mmaction2) - 开放式多媒体实验室的视频行为识别工具箱。
- [SlowFast Networks](https://github.com/facebookresearch/SlowFast) - 视频识别的官方实现。
- [Kinetics Dataset](https://github.com/cvdfoundation/kinetics-dataset) - 大规模视频数据集。

## Contributing

欢迎贡献！请参阅 [CONTRIBUTING.md](CONTRIBUTING.md) 了解详细信息。

### 贡献指南

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的改变 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request
